# 1. Download and install LM Studio from: https://lmstudio.ai/

#    - Search for one of these models:

#    - Download the model (choose Q4_K_M or Q5_K_M quantization for balance)
# 
# 3. Load the model:
#    - Go to the "Chat" tab
#    - Select your downloaded model from the dropdown
#    - Wait for it to load
# 
# 4. Start the local server:
#    - Go to the "Local Server" tab
#    - Click "Start Server"
#    - Verify it's running on http://localhost:1234
# 
# 5. Update CUSTOM_MODEL_NAME above to match your loaded model
# 
# 6. Run the POC scanner:
#    python run_poc_scan.py --target <url> --module xss_scanner
